数组指针与函数指针

Author: HanKangKai Date: 一月 8, 2024  21:32:37 Category: effectivemoderncpp

1.数组相关

int array[5] = {0,1,2,3,4};

array的类型为 int[5] 。

注：有指针数组，没有引用数组。

“Helloworld”字符串字面值为左值

2.数组名作为参数发生退化

int a[100]，int a[5]，int * a 都显示为**int ***类型。

（数组名发生退化）

int (a)[100]代表了 指向大小为100 的数组*的指针 包含数组大小。

3.函数相关

bool fun(int a, int b); ——————>**bool (int, int)**函数数据类型
*bool (funptr)(int a, int b); ———>bool (*) (int, int）函数指针类型
